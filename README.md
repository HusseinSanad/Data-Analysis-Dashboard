# Bike Sales Data Analysis (interactive Dashboard using MS Excel)
## project objectives 
The Bike Sales wants to know Who bought more bikes , the Owner of the bike sales understand their customers and grow more sales for future.

## Dataset Used 
-<a href="https://github.com/HusseinSanad/Data-Analysis-Dashboard/blob/main/Bike%20Project.xlsx">Dataset</a>

## Questions (KPI)
-Compare the sales and orders using pivot table 
- who purchased more -men or women?
- who income more – married or single ?
- which age brackets got the highest sales ?
- It shows who has traveled long distances from customers
- Relation between age and gender based in number of sales 
-highest selling region


- Dashboard interaction<a href=https://github.com/HusseinSanad/Bike-Dashboard/blob/main/Bike%20Bashboard.png> View Dashboard</a>

 ## Process
-Verify data for any missing values and anomalies.and sort out the same 
-Made sure data is consistent with and clean with respect to data tybe, data format and values used 
- Create pivot tables according to the questions asked 
-Merge all Pivot tables into one dashboard and apply sclicer to make dynamic

-Women customers are more likely to buy Products compared to men (~65%)
- The region of Pacific achieved higher sales
-The middle age brackets(31-55) is max contributing (~50%) and buys the most products
-Top 3 most purchased categories  of educations is Bachelor’s, Graduate Degree and High school
